Welcome to Paper.js for Saltarelle

This is a Paper.js metadata library for Saltarelle Compiler (it is a C# -> JS compiler, you can read more about it here).

Download

You can download the library at the left panel by clicking the "Download ZIP" button.

It contains two project:

  • PaperJs: the Saltarelle metadata library for Paper.js (generated code + some handwritten extensions)
  • PaperJsGenerator: metadata generator tool. Downloads the paper.js documentation and generates C# metadata source code from it.

About the repository

The project's Github repository is a fork of the SaltarelleCompiler's repository, because it contains also the other Runtime libraries (ex. Knockout, node.js, etc), so I just extended it.
